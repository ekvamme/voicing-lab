# Voicing Lab

A single-file app for learning jazz voice leading through inversions: play-along
sheet music (grand staff), a voice-leading graph showing each chord's inversion,
and a keyboard showing the left-hand chord shape — all advancing in sync at your
tempo. Includes Sweet Georgia Brown (1925, public domain) with full melody.

**Run it:** open `voicing-lab.html` — no build, no install. Needs internet for
piano samples and fonts (a synth fallback covers offline chords).

**On iPad:** open the GitHub Pages URL in Safari, then Share → *Add to Home
Screen*. It launches full-screen like a native app.

Docs: `DEV_NOTES.md` (architecture) · `PUBLIC_DOMAIN_STANDARDS.md` (which tunes
can legally get melodies) · `SONGS_TO_ADD.txt` (roadmap).
